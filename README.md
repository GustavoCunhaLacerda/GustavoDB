## 💻 Começando

### Requisitos

**Clonar o projeto na sua máquina e possuir o compilador gcc (ous similar) na máquina.**

```bash
$ git clone https://github.com/GustavoCunhaLacerda/GustavoDB.git
$ cd GustavoDB
```

**Siga esses passos**

```bash
# Compile o projeto
$ gcc ./db.c -o bin

# Inicie o programa
$ ./bin
```
